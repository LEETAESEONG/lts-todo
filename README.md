# 실행 방법

1. node 버전 확인 (v20.12.1)

```bash
node -v
```

2. 버전이 다른 경우

- [nvm설치 및 사용법](https://sound-programming.tistory.com/186)
- 해당 링크에서 nvm을 가지고 version을 v20.12.1로 맞춘다.

3. 버전이 같은 경우

- node_modules 설치

```bash
npm install
```

- 실행

```bash
npm run start
```
